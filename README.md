请使用chrome并在chrome快捷方式（可以专门专门建个用）属性 → 目标 “"C:\Program Files (x86)\Google\Chrome\Application\chrome.exe"” 后面添加  --disable-web-security （有空格）
这么设置是为了能够跨域，我是暂时没有别的办法。

然后随便起个服务器环境。

然后就可以看10档盘口了
